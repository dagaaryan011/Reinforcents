ReinforCents

Overview

ReinforCents is an immersive, simulation-based project that integrates finance and artificial intelligence into a functional version of a real-world stock and options market, populated by intelligent trading agents trained using multi-agent reinforcement learning techniques.

1. Stock & Options Market Simulation

The foundation of ReinforCents begins with the development of a Stock and Options Market Simulation - a digital trading exchange that supports live trading of financial instruments such as equities and derivatives.

Key features include:

  • Live order placement
  • Price/time-priority matching engine
  • Real-time trade execution
  • Dynamic order book management
  • Live analytics and market depth tracking

This component will replicate the core functionalities of modern electronic trading platforms, offering a realistic trading interface for agents to operate in.

2. Option Pricing Engine

The second phase introduces an Option Pricing Engine, enabling participants to understand and implement the theoretical models behind financial derivatives valuation.

This module will cover:

  • The Black-Scholes model for pricing European-style options
  • Integration of option prices into the trading environment for agent decision-making

By incorporating this engine, the simulation will allow agents to consider real-time valuation metrics in their trading strategies.

3. Multi-Agent Reinforcement Learning Environment

The core of the project is the creation of a Multi-Agent Reinforcement Learning (MARL) Environment — a simulated "battle arena" where intelligent trading agents learn and evolve through market interaction.

Participants will develop and train multiple AI agent types, including:

  • Market Makers: Provide liquidity by continuously quoting bid/ask prices
  • Retail Traders: Emulate behavior of trend-following, often irrational individual investors
  • Institutional Traders

These agents are trained using state-of-the-art reinforcement learning algorithms such as:

  • DDPG: Deep Deterministic Policy Gradient
  • SAC: Soft Actor-Critic
